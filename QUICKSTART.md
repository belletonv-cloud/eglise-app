# ⚡ Démarrage Rapide - Eglise App

## 🚀 3 façons de voir la démo

### 1. **Ultra Rapide** (10 secondes)
```bash
# Double-cliquez sur index.html
open index.html
```
→ Démo des aspirations avec animation

### 2. **Rapide** (30 secondes)  
```bash
npm run demo:quick
```
→ Serveur local + démo aspirations sur http://localhost:8080

### 3. **Complète** (2 minutes)
```bash
npm run demo
```
→ Application complète sur http://localhost:5173/demo-tour

## 🎯 Raccourcis Utiles

| Commande | Description |
|----------|-------------|
| `npm run demo` | Démo complète interactive |
| `npm run demo:quick` | Serveur pour index.html |
| `npm run setup` | Installation des dépendances |
| `npm run demo:build` | Build de production |

## 🔗 URLs de Démo

Une fois `npm run demo` lancé :

- **Tour guidé** : http://localhost:5173/demo-tour
- **Mode essai** : http://localhost:5173/ (cliquer "🎸 Essai sans compte")
- **Démo interactive** : http://localhost:5173/interactive

# Modules Standalone

Chaque module de l'application peut être extrait et utilisé comme une app indépendante.

## Music Stand — Application Standalone

```bash
cd frontend
npm run dev
# http://localhost:5173/music-stand
```

- Transposition ChordPro automatique
- Métronome intégré
- Auto-scroll
- Mode scène
- Navigation par setlist

## Check-in

```bash
cd frontend
npm run dev
# http://localhost:5173/checkin
```

- Scan de code ou recherche de membres
- Pointage pour événements et services
- Mode kiosque

## Planning

```bash
cd frontend
npm run dev
# http://localhost:5173/planning
```

- Calendrier des services
- Gestion des conflits de bénévoles
- Templates de plans
- Export PDF
